# CelikSense AI Upgrade Claude Code Pack

This ZIP contains Claude Code prompts, slash commands, SKILLS.md, plugin list, and step-by-step guide for the upgraded CelikSense AI project.

Included upgrades:
- Bilingual English/Bahasa Melayu
- ADHD Adaptive Reading Agent
- Dyslexia Adaptive Reading Agent
- Blind learner audio support
- Dashboard with multiple Agentic AI agents
- VS Code setup guide

How to use:
1. Extract this ZIP into your project folder.
2. Open the folder in VS Code.
3. Install recommended plugins in VS_CODE_PLUGINS.md.
4. Open Claude Code.
5. Use slash commands in .claude/commands or copy prompts from prompts folder.
