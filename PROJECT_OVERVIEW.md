# CelikSense AI: Agentic Multi-Sensory Learning Ecosystem

Tagline: Knowledge Without Barrier, Intelligence Without Limits.

CelikSense AI is an Agentic AI-powered inclusive learning ecosystem for deaf, blind, low-vision, ADHD, and dyslexic learners. It supports bilingual use in Bahasa Melayu and English, and uses multiple intelligent agents to personalize reading, provide accessibility support, monitor learning challenges, and recommend suitable interventions.

Core agents:
1. AI Librarian Agent
2. Reading Companion Agent
3. ADHD Adaptive Reading Agent
4. Dyslexia Adaptive Reading Agent
5. Accessibility Agent
6. Early Warning Agent
7. Intervention Recommendation Agent
8. AI Sign Language Agent

Current upgrade focus:
- Bilingual interface: Bahasa Melayu / English
- Blind learner audio guide and Text-to-Speech
- ADHD support: reading window, micro-reading, TTS, focus break, hyperfocus interest, highlighting, note builder
- Dyslexia support: OpenDyslexic-friendly mode, larger text, spacing, color overlay, TTS, audiobook mode, OCR-ready flow, mind map, voice reflection
