# STEP_BY_STEP_GUIDE.md

## Step 1: Open Project in VS Code
1. Open VS Code.
2. Open your CelikSense AI folder.
3. Make sure the folder contains index.html, login.html, signup.html, dashboard.html.

## Step 2: Install VS Code Plugins
Install:
- Live Server
- Prettier
- HTML CSS Support
- Auto Rename Tag
- JavaScript ES6 snippets
- Code Spell Checker

## Step 3: Add Claude Code Files
Copy this pack into your project folder.
Make sure the .claude/commands folder is inside your project.

## Step 4: Start With Homepage
Use /build-homepage to generate or improve:
- Hero section
- Login and Sign Up buttons
- 7 or 8 AI agent cards
- Bilingual selector

## Step 5: Build Login and Sign Up
Use /build-auth-pages.
Create simple prototype pages first.

## Step 6: Build Dashboard
Use /build-dashboard.
Dashboard should show:
- AI Librarian Agent
- Reading Companion Agent
- ADHD Adaptive Reading Agent
- Dyslexia Adaptive Reading Agent
- Accessibility Agent
- Early Warning Agent
- Intervention Recommendation Agent
- AI Sign Language Agent

## Step 7: Build Blind Audio Support
Use /build-blind-audio.
Add:
- Audio Guide
- Read Text Aloud
- Stop Audio

## Step 8: Build ADHD Agent
Use /build-adhd-agent.
Add:
- Reading Window
- Micro Reading
- TTS
- Focus Break
- ADHD Recommendation Engine

## Step 9: Build Dyslexia Agent
Use /build-dyslexia-agent.
Add:
- Dyslexia-friendly font
- Color overlay
- Larger text
- Line spacing
- TTS
- Mind map
- Dyslexia Recommendation Engine

## Step 10: Add Bilingual System
Use /build-bilingual-system.
All pages should allow language choice:
- English
- Bahasa Melayu

## Step 11: Test With Live Server
Right click any HTML file and select Open with Live Server.

## Step 12: Presentation Preparation
Use /build-presentation-content to generate:
- project pitch
- problem statement
- solution
- 8 agents
- impact
- roadmap
