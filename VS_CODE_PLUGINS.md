# VS_CODE_PLUGINS.md

Recommended VS Code plugins:

1. Live Server
Purpose: Run HTML pages locally in browser.

2. Prettier - Code formatter
Purpose: Make HTML, CSS, and JS neat.

3. HTML CSS Support
Purpose: Better HTML and CSS suggestions.

4. Auto Rename Tag
Purpose: Automatically rename paired HTML tags.

5. JavaScript (ES6) code snippets
Purpose: Faster JavaScript coding.

6. Code Spell Checker
Purpose: Check English spelling in project text.

Optional:
7. GitHub Copilot
Purpose: AI coding suggestions.

8. Claude Code
Purpose: Use prompt-driven development and slash commands.

9. Material Icon Theme
Purpose: Make file icons easier to see.
